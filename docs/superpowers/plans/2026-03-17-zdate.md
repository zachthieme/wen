# zdate Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a Go CLI that converts natural language dates to `yyyy-mm-dd` format.

**Architecture:** Single `main.go` wrapping `olebedev/when` for parsing. Three input modes (no-arg/TTY, positional arg, piped stdin) determined by argument count and TTY detection. One optional `--now` flag to override the reference date.

**Tech Stack:** Go, `olebedev/when`, `golang.org/x/term` (for TTY detection)

### Task 3: Implement no-arg mode (print today's date)

**Files:**
- Create: `main.go`

- [ ] **Step 1: Write minimal main.go**

Create `main.go` with flag parsing, TTY detection, and the no-arg mode:

```go
package main

import (
	"bufio"
	"flag"
	"fmt"
	"os"
	"strings"
	"time"

	"github.com/olebedev/when"
	"github.com/olebedev/when/rules/common"
	"github.com/olebedev/when/rules/en"
	"golang.org/x/term"
)

func main() {
	nowFlag := flag.String("now", "", "override reference date (yyyy-mm-dd)")
	flag.Parse()

	ref := time.Now()
	if *nowFlag != "" {
		parsed, err := time.Parse("2006-01-02", *nowFlag)
		if err != nil {
			fmt.Fprintf(os.Stderr, "error: invalid --now date %q, expected yyyy-mm-dd\n", *nowFlag)
			os.Exit(1)
		}
		ref = parsed
	}

	args := flag.Args()

	var input string
	switch {
	case len(args) > 0:
		input = strings.Join(args, " ")
	case !term.IsTerminal(int(os.Stdin.Fd())):
		scanner := bufio.NewScanner(os.Stdin)
		if !scanner.Scan() {
			fmt.Fprintln(os.Stderr, "error: failed to read from stdin")
			os.Exit(1)
		}
		input = strings.TrimSpace(scanner.Text())
	default:
		fmt.Println(ref.Format("2006-01-02"))
		return
	}

	w := when.New(nil)
	w.Add(en.All...)
	w.Add(common.All...)

	result, err := w.Parse(input, ref)
	if err != nil || result == nil {
		fmt.Fprintf(os.Stderr, "error: could not parse date %q\n", input)
		os.Exit(1)
	}

	fmt.Println(result.Time.Format("2006-01-02"))
}
```
